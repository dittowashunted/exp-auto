# **Discord Auto-React Selfbot**

An automated Discord selfbot that reacts to specific messages from targeted users. Perfect for entering giveaways automatically.

### ⚠️ WARNING

Using selfbots violates Discord's Terms of Service and can result in account termination. Use at your own risk. This project is for educational purposes only.

#### 

#### Features

🎯 Auto-react to messages from specific users

🔍 Filter by trigger text in messages or embeds

📊 Track reaction statistics

⏱️ Randomized delays to appear more human

💾 Persistent data storage

🎨 Clean terminal UI with real-time stats



#### Requirements

Node.js (v16 or higher)

npm



#### Installation



Clone this repository:



bashgit clone https://github.com/YOUR\_USERNAME/discord-auto-react.git

cd discord-auto-react



Install dependencies:



bashnpm install discord.js-selfbot-v13 prompt-sync



Run the bot:



bashnode bot.js
